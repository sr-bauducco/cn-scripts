# 🧮 Scripts de Cálculo Numérico | UnB (1/2026)

Repositório criado para guardar scripts desenvolvidos com o objetivo de auxiliar na verificação e validação de exercícios da disciplina de Cálculo Numérico da Universidade de Brasília (UnB), referente ao semestre 1/2026.

## 🚀 Sobre o Projeto
A ideia principal deste repositório é transformar os métodos numéricos teóricos vistos em sala de aula em código. Isso automatiza o processo de conferência de gabaritos das listas de exercícios e ajuda a consolidar o entendimento prático da estrutura e dos critérios de parada de cada algoritmo.

## 📂 Algoritmos Implementados

* [x] **Método da Bisseção** (`bissecao.py`): Script em Python que encontra raízes de funções contínuas através da divisão sucessiva de intervalos, exibindo uma tabela passo a passo com o erro a cada iteração.
* [ ] *Método de Newton-Raphson (em breve)*
* [ ] *Método das Secantes (em breve)*
